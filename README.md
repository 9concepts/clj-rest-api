# clj-rest-api

## Command

```bash
# Create application
lein new app clj-rest-api
```

```bash
# Run application (今の構成だと動かない)
lein run -m clj-rest-api.core

# Run application
clojure -M -m clj-rest-api.core

# Run application with hot reload
clojure -A:dev -M:dev
```
